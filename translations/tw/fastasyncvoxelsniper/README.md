## 關於這個插件

FastAsyncVoxelSniper 是知名但已停止維護的插件 "VoxelSniper" 的分支版本。  
FAVS 支援較新的 Spigot 和 Paper 版本，保留原版插件的所有功能，同時進行了優化和程式碼整理。
