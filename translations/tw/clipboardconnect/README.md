## 關於這個插件

Clipboard Connect 是一個動態的 Minecraft 插件，利用 Redis 讓多個伺服器之間的 WorldEdit 剪貼簿同步。這個工具簡化了在伺服器間傳送建築和結構的流程，方便 Minecraft 社群的協作與專案管理。
